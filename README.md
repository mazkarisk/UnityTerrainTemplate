# 説明

UnityのHDRP向けの、Terrainで使用できる草アセット群です。

# 利用方法

このリポジトリやAssetフォルダ内から必要なファイルをダウンロードしてください。

あるいは、以下のURLから元データをダウンロードする事も出来ます。リンク先のファイルも同じライセンスで利用できます。
